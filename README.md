## Workflow

**In Claude chat:**
1. "Update [client] [page] - [what you want to change]"
2. Paste GitHub token when prompted
3. Review changes in artifact
4. Confirm to commit and push

**Manual access:**
- Clone: `git clone https://github.com/amcreativemarketing/am-creative-client-sites.git`
- Or browse/edit directly on GitHub.com

## Team Access

Add collaborators: Settings → Collaborators → Add people
